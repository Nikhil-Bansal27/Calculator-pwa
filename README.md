# NIKHIL Calculator - PWA

🧮 A beautiful, accessible calculator PWA that works on all platforms!

## ✨ Features

- ✅ **Cross-Platform**: Works on Android, iOS, Windows, Mac, and Linux
- ✅ **Dark Mode**: Beautiful light and dark themes with persistent preference
- ✅ **Offline Capable**: Works even without internet connection
- ✅ **Installable**: Add to home screen on mobile or install on desktop
- ✅ **Calculation History**: Keep track of your calculations
- ✅ **Fast & Lightweight**: No ads, no bloat
- ✅ **Mathematical Functions**: Basic operations + percentage + square root

## 🚀 Live Demo

Visit the calculator here: [NIKHIL Calculator](https://nikhil-bansal27.github.io/calculator-pwa/)

## 📱 How to Download & Use

### On Mobile (Android & iOS)

1. **Open in Browser**
   - Go to: https://nikhil-bansal27.github.io/calculator-pwa/

2. **Install App**
   - **Android**: Tap the menu (⋮) → "Install app" or "Add to Home Screen"
   - **iOS**: Tap Share (↗) → "Add to Home Screen"

3. **Open Like a Native App**
   - The app will appear on your home screen
   - Open it anytime, works offline!

### On Desktop (Windows/Mac/Linux)

1. **Google Chrome or Edge**
   - Go to: https://nikhil-bansal27.github.io/calculator-pwa/
   - Click the install icon (looks like a download box) in the address bar
   - Click "Install"

2. **Open Like a Desktop App**
   - Find "NIKHIL Calc" in your applications
   - Pin to taskbar or start menu

3. **Firefox**
   - Bookmark the page to home screen for quick access

## 🎮 How to Use

| Button | Function |
|--------|----------|
| **C** | Clear all |
| **+, −, ×, ÷** | Basic operations |
| **%** | Percentage |
| **√** | Square root |
| **⌫** | Backspace |
| **=** | Calculate result |
| **🌙 / ☀️** | Toggle dark/light theme |

## 🔧 Installation for Development

```bash
# Clone the repository
git clone https://github.com/Nikhil-Bansal27/calculator-pwa.git

# Navigate to the folder
cd calculator-pwa

# Open index.html in your browser
# Or use a local server:
python -m http.server 8000
# Then visit http://localhost:8000
```

## 📝 File Structure

```
calculator-pwa/
├── index.html          # Main HTML file
├── manifest.json       # PWA configuration
├── service-worker.js   # Offline support
└── README.md          # This file
```

## 🎨 Customization

Edit `manifest.json` to change:
- App name and colors
- Start URL
- Display mode

Edit `index.html` styles to customize colors and layout.
